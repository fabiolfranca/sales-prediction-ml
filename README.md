## 📊 Previsão de Vendas com Machine Learning

### 🎯 Sobre o Projeto
Meu primeiro projeto de Machine Learning! Um modelo de Regressão Linear para prever vendas com base em investimentos em marketing nos canais: **TV, Rádio e Jornal**.

### 📈 Resultados Alcançados
- **R² = 0.899** - O modelo explica 89% da variação nas vendas
- **MAE = 1.46** - Erro médio de 1.46 unidades
- **MSE = 3.17**

### 🔍 Principais Insights
| Canal | Correlação | Impacto no Modelo |
|-------|------------|-------------------|
| 📺 TV | 0.78 | 0.0447 |
| 📻 Rádio | 0.58 | 0.1892 |
| 📰 Jornal | 0.23 | 0.0028 |

> 💡 **Descoberta importante:** *Rádio tem o **maior impacto por unidade investida**, mesmo tendo correlação menor que TV. Isso mostra que correlação e impacto real no modelo são coisas diferentes.*
